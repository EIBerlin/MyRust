# 🦀 MyRust

**MyRust** is a beginner-friendly Rust project created to explore and practice core Rust concepts.

---

## 📖 Description

This project serves as a personal playground for learning Rust.  

---

## 🚀 Getting Started

### 1. Install Rust

```bash
curl https://sh.rustup.rs -sSf | sh
```

### 2. Verify Installation

```bash
rustc --version
```

### 3. Create Project

```bash
cargo new MyRust
cd MyRust
```

### 4. Run the Project

```bash
cargo run
cargo run --bin rust_02_scope
```
